# 4. 啟動與測試

現在您可以啟動代理進行初步測試。

### 啟動 Web 介面
在 Cloud Shell 終端機中執行：

```bash
make run
```

### 在 Cloud Shell 預覽網頁
當終端機顯示啟動成功後：
1. 點擊 Cloud Shell 右上角的「網頁預覽」按鈕（圖示為視窗）。
2. 選擇「在通訊埠 8000 上進行預覽」。
3. 瀏覽器會開啟一個新分頁，顯示 AI 代理的對話介面。

### 試試看這些問題：
*   「你好，你是誰？」
*   「現在東京的時間是幾點？」（測試內建的時區工具）
*   「幫我寫一段 Python 程式碼。」
